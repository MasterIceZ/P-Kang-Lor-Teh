array : a[]={1,4,2,3,5,7,3,1}
n = 8
sorted : a[]={1,1,2,3,3,4,5,7}

upper_bound and lower_bound 
are use with sorted array
syntax :
idx = upper_bound(a,a+n,find)-a;
idx = upper_bound(a,a+n,find)-a;
*note find is number that want to find!
upper_bound of 4 is : 6
upper_bound use to find first index that (value > find)
lower_bound use to find first index that (value >= find)
